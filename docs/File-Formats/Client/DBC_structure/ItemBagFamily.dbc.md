### \`ItemBagFamily\` dbc

This dbc contains all bag families.

**Version is : 3.3.5a**

### Structure

| **Field Nb** | **Name** | **Type** |
|--------------|----------|----------|
| 1            | ID       | Int      |
| 2            | Name     | String   |

### Content

When including this template in another page, use parameter binary=1 to get binary values instead of ID, eg: 2^ID.
