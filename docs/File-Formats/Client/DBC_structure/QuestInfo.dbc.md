### \`QuestInfo\` dbc

This dbc contains all quest type that can be used in quest\_template.

**Version is : 3.3.5a**

### Structure

| **Field Nb** | **Name** | **Type** |
|--------------|----------|----------|
| 1            | ID       | Int      |
| 2            | Name     | String   |
