### \`QuestSort\` dbc

This dbc contains all quest sorts that can be used as negative values in the [ZoneOrSort](Quest_template#ZoneOrSort) field in [quest\_template](Quest_template).

**Version is : 3.3.5a**

| 1            | ID       | Int      |
| 2            | Name     | String   |
