### ItemSubClass.dbc

This DBC contains all items possible classes/subclasses combinations.

**Version is : 3.3.5a**

### Structure

| **Field Nb** | **Name**    | **Type** |
|--------------|-------------|----------|
| 1            | Class ID    | Int      |
| 2            | SubClass ID | Int      |
| 10           | Hand number | Int      |
| 11           | Name        | String   |
| 28           | Plural Name | String   |
