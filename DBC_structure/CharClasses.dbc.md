### \`ChrClasses\` dbc

This dbc contains all players possible classes.

**Version is : 3.3.5a**

### Structure

| **Field Nb** | **Name** | **Type** |
|--------------|----------|----------|
| 1            | ID       | Int      |
| 5            | Name     | String   |

### Content

When including this template in another page, use parameter binary=1 to get binary values instead of ID, eg: 2^(ID-1).

| Value | ID  | Name         |
| ----- | --- | ------------ |

| 2     | 2   | Paladin      |
| 4     | 3   | Hunter       |
| 8     | 4   | Rogue        |
| 16    | 5   | Priest       |
| 32    | 6   | Death Knight |
| 64    | 7   | Shaman       |
| 128   | 8   | Mage         |
| 256   | 9   | Warlock      |
| 1024  | 11  | Druid        |
